## Installation + starting

This project is tested with oTree version 5.8.5. and Python 3.10

Install requirements:
```bash
pip install -r requirments.txt
```

Start development server:
```bash
otree devserver
```

## Read from CSV

CSV contains:

- all questions with 5 different multiple choice options
- solution
- quiztype
- image reference

The CSV is TAB-seperated!
